# Comparison of Task-Modulated Functional Connectivity (TMFC) Methods

Repository for code and simulations from  <br/> 
**"Comparison of whole-brain task-modulated functional connectivity methods for fMRI task connectomics"** <br/>
by Masharipov, R., Knyazeva, I., Korotkov, A., Cherednichenko, D. &amp; Kireev, M.

Use the repository [Discussions](https://github.com/Masharipov/TMFC_simulations/discussions) for questions or email masharipov@ihb.spb.ru.

## Overview

Here, we provide:

1. [Task design files](https://github.com/Masharipov/TMFC_simulations/task_designs) (.*mat format) containing stimulus onsets, durations, condition names, and weighting factors for synaptic matrices. Onsets, durations and names are defined in the same way as for **multiple conditions** *.mat file for SPM12.
  
2. [Python code](https://github.com/Masharipov/TMFC_simulations/python_code) for TMFC simulations based on **large-scale Wilson-Cowan neural mass model** and **Ballon-Windkessel haemodynamic model**.

3. User-friendly [Jupyter notebooks](https://github.com/Masharipov/TMFC_simulations/jupyter_notebooks) for reproducing our simulations.

4. [Simulated BOLD time series files](https://github.com/Masharipov/TMFC_simulations/simulated_BOLD_time_series) (*.mat format) for all experiments presented in the paper.

5. [MATLAB code](https://github.com/Masharipov/TMFC_simulations/matlab_code) for TMFC analysis using:
    * correlation difference approach (**CorrDiff**),
    * standard psychophysiological interactions (**sPPI**),
    * generalised psychophysiological interactions (**gPPI**),
    * correlational psychophysiological interactions (**cPPI**),
    * beta-series correlations based on least-squares-all approach (**BSC-LSA**),
    * beta-series correlations based on least-squares-separate approach (**BSC-LSS**).
